# NovelpiaDownloader

[English](README_en.md)

노벨피아에서 소설을 다운로드하여 **EPUB** / **TXT** 형식으로 저장하는 도구입니다.

## 다운로드

[Releases](https://github.com/CjangCjengh/NovelpiaDownloader/releases)에서 `NovelpiaDownloader.exe`를 받으세요.

## 사용 방법

1. `NovelpiaDownloader.exe`를 실행합니다.
2. 화면 상단의 언어 드롭다운에서 표시 언어를 선택합니다.
3. **로그인** 그룹에서 로그인 방식을 선택합니다.
   - 이메일 계정: 이메일과 비밀번호를 입력하고 **로그인** 버튼을 누릅니다.
   - SNS 계정: 브라우저 개발자 도구 등으로 LOGINKEY 값을 확인한 뒤 LOGINKEY 칸에 입력하고 **로그인** 버튼을 누릅니다.
4. 필요에 따라 **스레드 / 간격 (초) / 재시도**에서 병렬 수와 재시도 횟수를 조정합니다.
5. 왼쪽 아래 **다운로드** 그룹에서 다음을 설정합니다.
   - **소설 번호**: 노벨피아 소설 URL 또는 소설 번호를 입력합니다.
   - **화부터 / 화까지**: 체크하면 범위를 지정합니다. 체크하지 않으면 전체 화를 다운로드합니다. 본문 완결 후의 BONUS 화는 마지막 EP 번호 다음 번호로 간주되어 **화까지** 범위에 포함됩니다. 예: 마지막 EP가 100이면 **화까지** 101 → BONUS 1화, 102 → 2화.
   - **BONUS 제외 / BONUS 항상 포함**: BONUS 화 다운로드를 강제합니다. **제외**는 범위와 무관하게 BONUS을 내리고, **항상 포함**은 범위와 무관하게 모든 BONUS을 받습니다. 둘 다 체크하지 않으면 위 범위 규칙을 따릅니다.
   - **저장 경로**: 출력 폴더를 지정합니다.
   - **형식**: EPUB / TXT 중 선택합니다.
6. **다운로드** 버튼으로 즉시 다운로드하거나, **목록에 추가**로 여러 작품을 큐에 넣은 뒤 우측 목록에서 **다운로드** 버튼으로 일괄 실행합니다.
